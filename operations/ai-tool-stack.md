## Delivery Workflow by Service

### Website Build (1-2 weeks)
```
Day 1-2: Discovery + v0 prototype
Day 3-5: Cursor development + content
Day 6-7: Supabase integration (if needed)
Day 8-10: Vercel deployment + testing
Day 11-14: Revisions + handoff
```

### AI Marketing (Ongoing)
```
Weekly:
- Claude: Generate content calendar
- Midjourney: Create visuals
- Canva: Format for each platform
- Buffer: Schedule posts
- Make.com: Auto-publish + track
Monthly:
- Analytics report (Claude analysis)
- Strategy refinement
```

### Custom App (2-4 weeks)
```
Week 1: Requirements + Lovable/Bolt prototype
Week 2: Cursor development + database setup
Week 3: Feature completion + testing
Week 4: Deployment + documentation
```

## Margin Calculation Example

**Website Project**: $3,500 client fee
- AI tools (amortized): ~$200
- Your time: 20 hours × $100/hr = $2,000
- **Net profit**: $1,300 (37% margin)

**Marketing Retainer**: $2,000/month
- AI tools: ~$300
- Your time: 10 hours × $100 = $1,000
- **Net profit**: $700/month (35% margin)

**Target**: 30-40% margins while delivering faster than competitors.
