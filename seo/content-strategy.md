## SEO Content Best Practices

### On-Page Optimization Checklist

For each blog post:

- [ ] Title tag includes primary keyword (under 60 chars)
- [ ] Meta description includes keyword + CTA (under 160 chars)
- [ ] H1 includes primary keyword naturally
- [ ] URL slug is keyword-focused
- [ ] At least 1,500 words for pillar content
- [ ] Internal links to relevant service pages
- [ ] External links to authoritative sources
- [ ] Alt text on all images
- [ ] Schema markup (Article or FAQ)
- [ ] Social sharing images configured
